# GeoLens

GeoLens is a study project developed at the Institute for Geoinformatics at the University of Münster. It addresses the growing challenge of disinformation and manipulated geomedia, especially online images shared with misleading location, time, or context.

The project aims to develop a browser extension that helps users verify geomedia directly while browsing. GeoLens supports the analysis of metadata, timestamps, location information, and external geographic sources such as OpenStreetMap and satellite-based data.

GeoLens is intended as a technical prototype to support more transparent and trustworthy interpretation of visual content in areas such as journalism, science, civic documentation, and public awareness.
